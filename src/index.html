The GPAPI Package Demo

The main GoodPractice API is a fully secured RESTful API that gives client access to the structure, features and content of any Subscription for which they have access

The GPAPI package is a helper module that greatly simplifies the security handshake process and then any subsequent access to the main GoodPractice API endpoints

This demo app is intended to get you started using the GPAPI helper package. The demo defines its own simple facade demo API with a few example end-points which in turn call out to the main GoodPractice API, via the GPAPI package, and return the results.

Each of the demo end-points is shown below but of course the main GoodPractice API supports many more features than this. You can see the complete documentation here GoodPractice API Full Documentation

Test the Demo EndPoints

Retrieve Subscription Details

http://[domain:port]/gp/subscription/[subscription-url]

 

Retrieve the Top Level 'root' Folders

http://[domain:port]/gp/[content-model-id]/root-folders

 

Retrieve a List of Child Folders for a Specified Parent Folder

http://[domain:port]/gp/folder/[parent-folder-id]

 

Retrieve a List of Content Documents for a Specified Folder

http://[domain:port]/gp/[folder-id]/documents

 

Retrieve a Specified Content Document

http://[domain:port]/gp/content/[type]/[content-document-id]
